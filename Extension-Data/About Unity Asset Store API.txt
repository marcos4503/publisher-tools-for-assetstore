This file shows some APIs of Unity and how to use each.

[Sales API]

URL:            https://publisher.assetstore.unity3d.com/api/publisher-info/sales/(Publisher_ID)/(Period).json
Publisher_ID:   It can be found at the URL of your Unity Publisher page. 
                Example: https://assetstore.unity.com/publishers/12345 <- Here is the ID
Period:         Period of Sales
                Example: 202103 <- 2021 year, 03 month

[Reviews RSS]

https://publisher.assetstore.unity3d.com/feed/publisher-name/key34DWFDS344fsdw345453DFsw3435/activity.rss

This RSS URL (can be found in the "Reviews" tab of your Publisher Portal) contains all revisions published by
users in their Assets.
